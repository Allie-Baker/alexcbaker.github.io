This is a 3-D printed model of Niagara Falls! I chose this one because it is a beautiful land mark in Canada and it looked cool as a 3D printed model. I had fun getting it up on Sketchup. I used the Geo-spatial Data with Terrain2STL to obtain the coordinates because SketchUp can process the longitude and latitude so it was easy to do it.
